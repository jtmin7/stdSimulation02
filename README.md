### 명세서  

클래스: 지원자, 아파트, 렌덤추첨  
구조체: 개인정보  

#### 기능
* 지원자  
  * 맴버:  
    * 백터(개인정보)  vec_목록_지원자개인정보
    * 백터((평수별지원자그룹)  vec_목록_평수별지원자그룹
  * 기능:  
    * 동적배열로 개인정보를 추가할 수 있다  
    * 테스트용 개인정보 자동생성  
    * 지원자의 개인정보 오름차순(평수별, 가산점) 출력  
    * 희망평수별 그룹화(희망평수, 지원자순번)  
  ---
      * 클래스1 (평수별지원자그룹)  
        * int 희망평수  
        * 백터1(가점)  vec_목록_가점
        * 백터2(클래스2)  vec_목록_가산점그룹
        * 백터3(int) 지원자순번s  
        * 함수(지원자순번을 돌며 가점을 체크하고 백터1,2를 추가하고 오름차순에 따라 배열한다(앞에 수보다 작으면 앞으로 크면 그대로)  
      * 클래스2 (가산점그룹)  
        * int 가산점
        * 백터(int) vec_list_지원자순번 
  ---
    * 평수그룹 가점그룹 출력  

* 아파트  
  * 맴버:  
    * 백터(아파트정보) 아파트정보s 
  * 기능:  
    * 아파트 동, 호수로 아파트정보 자동생성  
    * 평수 별 아파트의 순번 할당
    * 아파트 동, 호수, 평수 출력  
    * 아파트 평수별 갯수 계산, 출력  

* 렌덤추첨 
  * 맴버:  
    * 백터(평수별지원자그룹) 평수그룹_지원자 
    * 백터(평수별아파트그룹) 평수그룹_아파트  
    * 아파트  
    * 지원자
  * 기능:  
    * 아파트와 지원자를 생성자의 인수로 넣는다  
    * 렌덤추첨의 평수그룹_지원자(당첨순위지정) 지정  
      1. 지원자의 평수그룹에서 가산점들을 돌며 같은 가산점그룹에 속한 지원자 순번이 2개 이상일 때 랜덤 함수를 이용해 추첨한다  
      1. 추첨 순번에 따라서 렌덤추첨의 평수그룹_지원자 의 지원자순번s 을 할당한다  
    * 아파트의 아파트정보s 를 돌면서 평수그룹_아파트 에 아파트순번을 추가하거나 평수그룹_아파트 자체를 추가한다  
    --- 
      * 백터1(평수별아파트그룹) 평수그룹_아파트
      * 클래스1 (평수별아파트그룹)  
        * int 평수  
        * 백터1(int) 아파트순번s  
        * int count (이 평수에 속한 총 세대 수)
    ---  
    * 평수그룹_아파트 의 아파트 아파트순번s 에 순번 과 같은 아파트 의 아파트정보s 에 당첨자순번을 순서에 따라 할당한다  

* 개인정보: 지원자이름, 전화번호, 주소, 희망평수, 가산점, 순번  
* 아파트정보: 아파트동, 아파트호, 평수, 층수, 순번, 당첨자순번

* 클래스1 (평수그룹)  
  * int 희망평수  
  * 백터1(가점)  
  * 백터2(클래스2)  
  * 백터3(지원자순번)  
  * 함수(지원자순번을 돌며 가점을 체크하고 백터1,2를 추가하고 오름차순에 따라 배열한다(앞에 수보다 작으면 앞으로 크면 그대로)  
* 클래스2 (가산점그룹)  
  * 백터(가산점에 따른 지원자순번)
